Project Modules

This project has declared the following modules:

Name Description
gae Dependencies for Google App Engine webapp deployment including native datastore access.
hibernate Runtime dependencies for Hibernate JPA implementation
xuml-diagrams-core Diagram generation core classes
xuml-diagrams Generates javascript browser deployed dynamic diagrams from xml
xuml-model-compiler Java model compiler including JPA generation from xml definitions.
xuml-model-compiler-runtime Runtime classes for use with classes generated by xuml-model-compiler
miuml-jaxb Contains miUML metamodel schema and xuml-tools extensions schema and generates JAXB classes from those schemas.
xuml-tools-maven-plugin Maven plugin for xuml-tools
xuml-model-archetype A collection of tools for Executable UML in the style of Mellor and Balcer and miUML.org based on JPA and Akka
order-tracker A collection of tools for Executable UML in the style of Mellor and Balcer and miUML.org based on JPA and Akka
order-tracker-webapp Web application for order-tracker using REST API
state-diagram-viewer Swing app for viewing state diagrams
dummy Dummy project required for release process in travis
xuml-model-compiler-test Unit tests for xuml-model-compiler
